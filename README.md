# Telegram Rcon LOGGER 
- Установка зависимости: 
```bash
   py -m pip install -r requirements.txt
```

1. `cfg.py` - параметры
2. `main.py` - запуск тг бота вместе с логгером.
Получить chat_id и thread_id (id топика) - `/start`
